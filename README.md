# Dashboard Académico React

## Descripción

Dashboard académico desarrollado con React que permite gestionar materias y visualizar el rendimiento académico de un estudiante.

## Funcionalidades

* Agregar materias con su respectiva nota.
* Eliminar materias.
* Calcular automáticamente el promedio general.
* Mostrar el estado académico:

  * Excelente ⭐
  * Aprobado 👍
  * En Riesgo ⚠️
* Validación de datos de entrada.
* Persistencia de datos mediante Local Storage.
* Actualización dinámica de la interfaz usando React Hooks.

## Tecnologías utilizadas

* React
* JavaScript
* Vite
* HTML
* CSS Inline Styles

## Instalación

1. Clonar el repositorio:

```bash
git clone https://github.com/samuk24-hash/DashboardParaEstudiantes
```

2. Entrar al proyecto:

```bash
cd DashboardParaEstudiantes
```

3. Instalar dependencias:

```bash
npm install
```

4. Ejecutar el proyecto:

```bash
npm run dev
```
